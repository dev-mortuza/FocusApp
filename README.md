# FocusApp

## Project Overview
FocusApp is designed to help users improve their productivity by managing tasks and tracking work sessions.

## Features
- Task management
- Timer for work sessions
- Analytics for user activity
- User-friendly interface

## Installation
Clone the repository and install the dependencies:

```bash
git clone https://github.com/dev-mortuza/FocusApp.git
cd FocusApp
npm install
```
